## 运行方法

### 方法1：在代码目录下运行（推荐）

```bash
cd 实验/第三章/代码
conda activate ml
python 作业1_直方图均衡化.py
python 作业2_拉普拉斯锐化.py
python 作业3_位平面分析.py
```

### 方法2：从项目根目录运行

```bash
conda activate ml
python 实验/第三章/代码/作业1_直方图均衡化.py
python 实验/第三章/代码/作业2_拉普拉斯锐化.py
python 实验/第三章/代码/作业3_位平面分析.py
```
