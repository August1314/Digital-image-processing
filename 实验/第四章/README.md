## 算法说明

### 二维FFT

使用NumPy的FFT实现：
```python
F(u,v) = FFT2D[f(x,y)]
f(x,y) = IFFT2D[F(u,v)]
```

### 高斯低通滤波器

公式：`H(u,v) = exp(-D²(u,v) / (2*D0²))`

其中：
- D(u,v) 是到频谱中心的欧氏距离
- D0 是截止频率（标准差）
- 能量保留率 = Σ|F_filtered|² / Σ|F_original|²
